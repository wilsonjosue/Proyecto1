CURSO: Programación Web I

Este laboratorio se elaboro consiste en crear un conjunto de páginas funcionamiento similar que una wikipedia. La idea es que un usuario pueda crear nuevas páginas usando un formulario y un lenguaje conocido como markdown. Que nos permite escribir páginas web sin saber HTML, en este laboratorio se tendrá que usar expresiones regulares para traducir un subconjunto del lenguaje markdown
